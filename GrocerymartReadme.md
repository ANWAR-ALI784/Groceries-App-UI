


# 🛒 GroceryMart - Flutter Grocery Shopping App


 ✅ Features

- 🏠 Home screen with grocery products
- ℹ️ Product detail screen with image animation
- ➕ Add to Cart button with badge update
- 🛒 Cart screen with total and remove option
- 🚀 State management using Provider
- 📱 Mobile-friendly design

---

# Tech Stack

- Flutter (Latest Stable)
- Provider (State Management)
- Material UI
- Dart
